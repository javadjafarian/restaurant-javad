# 🍽️ رستوران جواد

یک وب‌سایت رستوران با جنگو

## ✨ امکانات

- 🔐 احراز هویت (ورود/ثبت‌نام/خروج)
- 👤 پروفایل کاربر با ویرایش اطلاعات
- 🛒 سیستم سفارش غذا
- 📞 فرم تماس با ما
- 🍳 صفحات پویا (صبحانه، ناهار، شام، فست‌فود، ایرانی، نوشیدنی)
- 🔢 نمایش اعداد فارسی
- 📱 طراحی ریسپانسیو
